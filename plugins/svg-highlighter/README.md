# @svg-shards/highlighter

Highlight SVG shards one at a time. Optional zoom/pan/rotate viewport.

Peer dependencies: `svg-shards` ^0.2.0, `@preact/signals-core` ^1.14.2

## Playground

```bash
# From repo root — plugin playground at :3000
npm run demo

# From this package — plugin playground at :3002
npm run playground:dev
```

Plugin lessons are also embedded in the core playground under **Plugins → svg-shards/highlighter**.

See [docs/plugins/svg-highlighter.md](../../docs/plugins/svg-highlighter.md) for API details.
